About Historical Weather Data

  1. 2-3 years of hourly data from the Meteorological Department
  2. Wind speed at 100m, 70m, 10m, Wave-height, Visibility
  3. Missing time steps
  4. Invalid readings like -999

Tidying the Data

  1. Correlation found between different variables
  2. Finding outliers - using boxplot using Inter-Quartile Range = 3
  3. Linear interpolation for missing values, invalid values & outliers
  4. Check stationarity of the time series - running window of mean & variance

Modeling Techniques

Weather forecast should be stochastic to make error estimates

  1. ARIMA - poor long term prediction, lacks periodicity
  2. Neural Network - black box, cannot differentiate summer/winter
  3. Fourier Transform (periodic) + Markov process (stochastic)
